# Generate tooltips for interactive plot elements

Generate tooltips for interactive plot elements

## Usage

``` r
generate_tooltip(dataset, tooltip_vector)
```

## Arguments

- dataset:

  \`\[data.frame\]\` A data frame containing the variables to be used in
  tooltips

- tooltip_vector:

  \`\[character(n)\]\` A named character vector where names are labels
  and values are column names

## Value

A character vector of HTML-formatted tooltip strings
