# Changelog

## dv.swimmerplot 1.3.0

- Updated module to return dataset names for dv.manager filters
- Changed the font of plot to Liberation Sans
- Adjusted plot height to ensure full-width display
- Fixed incorrect sorting direction when subjects are grouped

## dv.swimmerplot 1.2.0

- Added local filtering support.
- Added support for text annotations.

## dv.swimmerplot 1.1.0

- Complete reimplementation of the visualization engine
- Migrated from plotly to ggplot2 and ggiraph for interactive
  visualization

## dv.swimmerplot 1.0.0

- First version of dv.swimmerplot
- Interactive visualization using plotly
